A native **Safari Web Extension** for **Twitch** and **Kick** across macOS, iOS, and iPadOS. All third-party emotes rendered inline (BTTV, FFZ, 7TV). Auto-claims channel points, Drops, Moments, and watch streaks. Split chat, anonymous viewing, Picture-in-Picture, forced video quality, tab-complete carousel, and 60+ more features. Zero accounts, zero telemetry, zero tracking.

## Features

- **Full BTTV, FFZ, and 7TV support.** Every third-party emote rendered natively on both Twitch and Kick, with real-time updates as channels add or remove them. Animated emotes can be toggled on or off.
- **Auto-claim everything.** Channel points, Drops, Moments, and watch streaks collected automatically across every channel you watch. Survives tab sleep and Safari restart.
- **Tab-complete carousel.** Type a partial emote name, press Tab to cycle through inline previews. Works for native Twitch emotes, BTTV, FFZ, and 7TV alike.
- **Split chat.** Alternating message backgrounds in 11 color themes for easier reading on fast-moving chats.
- **Picture-in-Picture.** One-click PiP button right in the player, on both Twitch and Kick.
- **Force video quality.** Lock any quality from 160p to Source. Persists across channel navigation.
- **Anonymous viewing.** Watch streams without appearing in the viewer list.
- **OLED mode.** Pure-black background for OLED displays with a transparent chat overlay.
- **First-time chatter highlights.** Spot new chatters instantly with colored backgrounds.
- **Deleted message display.** Deleted messages survive as strikethrough text so conversation context isn't lost.
- **Mention highlights and keyword filters.** Mention sound, title flash, color-coded mentions, full keyword block and highlight rules.
- **Spam filter.** Reduces zero-width emote spam, repetition, and ALL-CAPS automatically.
- **Chat search.** Find messages across the visible chat history.
- **User pronouns.** PronounDB integration in chat lines.
- **Custom nicknames.** Set display names for any user, per channel.
- **Enhanced user cards.** More info at a glance when clicking usernames.
- **Auto theater mode.** Enter theater mode automatically when you open a stream.
- **33 languages.** Fully localized including settings UI and onboarding. RTL layout for Arabic and Hebrew.

## What makes Purple Crow special

🎮 **Native on iOS, iPadOS, AND macOS.** Most Twitch and Kick enhancers are browser-extension-only or desktop-only. Purple Crow runs in Safari on every Apple device, with the same settings, the same auto-claim, the same emotes everywhere.

🔥 **Both Twitch AND Kick.** Most Twitch tools ignore Kick. Most Kick tools ignore Twitch. Purple Crow ships every feature on both platforms where it's technically possible, with a parity ledger in the settings schema so no feature gets forgotten on the other side.

⚡ **Tab-complete that actually previews.** Native Twitch tab-complete shows one suggestion at a time and rewrites your input on every press. Purple Crow shows an inline preview row of the next 6 to 10 candidates so you can see what you're picking before you commit.

💎 **Auto-claim that survives sleep.** Channel points, Drops, Moments, watch streaks, all of them, automatically, across all your watched channels. The auto-claim engine survives tab sleep, Safari restart, and even Mac sleep where Safari permits.

🎨 **Split chat, 11 themes.** Alternating-row backgrounds in 11 color schemes. After an hour of split chat, the default Twitch chat reads like an unstyled HTML page.

🌐 **33 languages, fully translated.** Including the in-app settings, the onboarding, and the GitHub Pages site. RTL layout flip for Arabic and Hebrew. The language picker is one click away.

🛡️ **Built for privacy.** Zero accounts, zero telemetry, zero analytics. All settings stored locally on your device. The only network requests Purple Crow makes are to BTTV, FFZ, and 7TV, the emote providers themselves, on channels you're actively watching.

## Built with

**Safari Web Extension MV3** · **WebKit** · **SwiftUI** · **BTTV API** · **FrankerFaceZ API** · **7TV EventAPI** · **PronounDB**

## Privacy

No accounts, no analytics, no telemetry, no tracking. Purple Crow never phones home. The only network requests are to BTTV, FFZ, and 7TV APIs to fetch the emotes themselves, and those happen only on channels you're actively watching. All settings, custom nicknames, and channel preferences live in `browser.storage.local` on your device.
